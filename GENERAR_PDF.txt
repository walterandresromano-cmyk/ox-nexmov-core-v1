## Descripción

Este paquete contiene una presentación profesional e institucional para ofrecer la plataforma oX NEXMOV a dealers con acceso gratuito por período limitado.

### Archivos Incluidos

1. **PRESENTACION_DEALERS.html** - Presentación completa de 12 páginas en HTML
2. **PRESENTACION_DEALERS.md** - Contenido en markdown (referencia)
3. **GENERAR_PDF.txt** - Instrucciones para convertir a PDF (este archivo)

---

## ✅ Cómo Generar el PDF

### Opción 1: Navegador (Recomendado - Resultado Perfecto)

**Pasos:**

1. **Abrir el HTML en tu navegador:**
   - Haz doble clic en `PRESENTACION_DEALERS.html`
   - O arrastra el archivo a tu navegador favorito (Chrome, Firefox, Edge, Safari)

2. **Generar PDF:**
   - **Windows/Linux:** Presiona `Ctrl + P`
   - **Mac:** Presiona `Cmd + P`

3. **Configurar impresión:**
   - En el diálogo de impresión:
     - Destino: **"Guardar como PDF"**
     - Márgenes: **Normal**
     - Escala: **100%**
     - Fondo: **Habilitar gráficos de fondo**
     - Papel: **Carta (8.5" × 11")**

4. **Guardar:**
   - Nombre: `PRESENTACION_DEALERS.pdf`
   - Ubicación: La carpeta actual del proyecto
   - Presiona **Guardar**

✨ **¡Listo!** Tendrás un PDF de 12 páginas perfectamente formateado.

---

### Opción 2: Línea de Comandos (Node.js)

Si tienes Node.js instalado:

```bash
# Instalar html-pdf
npm install html-pdf

# Generar PDF
node generate-pdf.js
```

**Requisitos:** Node.js v14+

---

### Opción 3: Python (WeasyPrint)

Si tienes Python 3.6+:

```bash
# Instalar WeasyPrint
pip install weasyprint

# Generar PDF
python generate-pdf.py
```

**Requisitos:** Python 3.6+, pip
